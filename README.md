# Farm-Management-System-ETL
A data analytics and visualization project developed using Microsoft Power BI to analyze agricultural productivity and smart farming factors such as crop yield, rainfall, soil moisture, NDVI index, irrigation methods, and fertilizer usage.

Project Overview:-
This dashboard helps in understanding how environmental and farming conditions affect crop production. The project uses interactive visualizations and KPI indicators to provide insights into:

* Crop-wise yield analysis
* Region-wise performance
* Rainfall and temperature impact
* Soil moisture monitoring
* NDVI-based vegetation analysis
* Irrigation and fertilizer effectiveness
* Crop disease monitoring


Features:- 
* Interactive KPI Cards
* Dynamic Filters and Slicers
* Crop Yield Analysis
* Weather Impact Visualization
* Soil Health Monitoring
* Region-wise Dashboard Insights
* Smart Farming Analytics
  
KPIs Used:-
* Total Yield
* Average Yield
* Average Temperature
* Total Rainfall
* Average Soil Moisture
* Average NDVI
* Disease Cases

Dataset Columns:-
* yield_kg_per_hectare
* crop_type
* region
* temperature_C
* rainfall_mm
* soil_moisture_%
* NDVI_index
* irrigation_type
* fertilizer_type
* crop_disease_status

Technologies Used:-
* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Star Schema
* Data Visualization


Dashboard Components :

* KPI Cards:-
Total Yield ,

* Charts:-

* Filters:-
Crop Type ,

Objective:-
The objective of this project is to use smart farming data to generate actionable insights that can improve agricultural productivity and decision-making through interactive business intelligence dashboards.

Conclusion:-
This project demonstrates how business intelligence and data analytics can support smart agriculture by identifying patterns, monitoring crop health, and improving farm management decisions.
